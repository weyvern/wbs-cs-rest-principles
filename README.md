## Install

- Fork project
- Clone your fork:

```bash
git clone <link-to-project>
cd <project-directory>/
npm install
```

# Commands

## Dev

Dev commands runs app with Node and the experimental flag --watch, you will need Node 18.11+

```bash
npm run dev
```

## Start

Start commands runs app with Node

```bash
npm start
```
